# ML Visualization Tool for Healthcare

## Project Description
A web-based tool for exploring healthcare datasets, training ML models, and visualizing performance across 20 clinical domains.

## Tech Stack
- Frontend: React 18 + Vite
- Backend: FastAPI
- ML Engine: scikit-learn, imbalanced-learn (SMOTE)

## Repository Structure
/frontend  → React application  
/backend   → FastAPI + ML models  
/docs      → Architecture & documentation  

## How to Run (Development)
### Frontend
cd frontend  
npm install  
npm run dev  

### Backend
cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload
